# Multi-Agent ShrutiBuilder PPT Generator 🤖

An advanced, state-of-the-art multi-agent framework orchestrating 12 AI agents to automatically analyze ShrutiBuilder PDFs, perform market research, invent a solution, design the system architecture, build a presentation deck, generate architecture diagrams, and export everything to professional formats (.PPTX and .PDF).

## 🚀 Features

- **End-to-End Orchestration**: Uses `LangChain` & `LangGraph` for complex sequential logic.
- **Agent Intelligence**: Employs Google Gemini `2.5-pro` for deep analytical product and consulting-level ideation.
- **Native Generation**: Builds `.pptx` natively without relying on web placeholders.
- **Smart Formatting**: Converts the PowerPoint immediately to `.pdf` seamlessly (Requires Microsoft Office Installed on Windows).
- **Automated Architect**: Injects Mermaid-based diagrams directly generated by AI into the final PPT framework.

## 🛠 Prerequisites

1. Create a python virtual environment, activate it.
   ```bash
   python -m venv venv
   .\venv\Scripts\activate
   ```
2. Install required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Update `.env.example` -> `.env` with your actual Google Gemini API key.

## 💻 Usage

Run the main application:
```bash
python main.py
```

### Guided Prompts:
You will be prompted to:
1. Provide the **ShrutiBuilder Topic / Subject**.
2. Optional: Provide a path to a **ShrutiBuilder Requirements PDF** (local path).
3. Specify any **Slide limits** configured.
4. Pass any **Special instructions** to inform the judges.

Wait for the 12 Agents to process... Your presentation artifacts will be constructed and saved inside the `/output/` folder and logs stored internally inside `/logs/`.
